# 基于moshuiD/Colorful-Keyborad-Led-Color-Setting的修改
1.去掉启动对话框，启动默认RGB循环，默认最小化到托盘。

2.支持启动参数，启动参数是RGB循环速度1-7，默认是1。快捷方式路径后加空格加参数就可以按参数启动RGB循环速度。

3.修改程序图标。

4.添加托盘按钮，点击后可最小化到托盘。
